<!-- shortDescription:begin -->

Browser-mediated communication via the user's agent.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Interactions where data transits the user's browser, e.g., OAuth/OIDC authorization redirects and SAML HTTP-Redirect/POST bindings. Useful for user interaction and consent, but subject to browser policies, URL length/visibility, and referrer leakage if not carefully designed.

<!-- longDescription:end -->
