<!-- shortDescription:begin -->

Long-lived credential to obtain new access tokens.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Longer-lived credential used to obtain new access tokens without user interaction. Must be kept confidential, is commonly rotated on use, and can be revoked when compromised or no longer needed.

<!-- longDescription:end -->
