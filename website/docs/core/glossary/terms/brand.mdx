<!-- shortDescription:begin -->

Per-domain settings for UI,default flows,and behavior.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

A Brand applies visual identity and behavior to requests on a domain or wildcard. It controls title, logo, favicon, theme, default flows (authentication, logout, recovery, user settings, device code), default application redirects, and global attributes like locale.

<!-- longDescription:end -->
