termName: Identity Provider (IdP)
---

<!-- shortDescription:begin -->

Authority that authenticates users and issues assertions/tokens.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

In SAML, the IdP issues assertions. In OIDC, this role is fulfilled by the OpenID Provider (OP), which is also the Authorization Server (AS) in OAuth2 terms; it authenticates users, obtains consent, and issues tokens.

<!-- longDescription:end -->
