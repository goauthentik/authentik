<!-- shortDescription:begin -->

Access‑Request/Accept/Reject and Accounting messages.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Core flow: NAS sends Access‑Request; server replies Access‑Accept (optionally with attributes) or Access‑Reject. Accounting‑Start/Stop records session usage.
