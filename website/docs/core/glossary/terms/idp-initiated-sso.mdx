<!-- shortDescription:begin -->

SSO flow started at the Identity Provider.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

User launches from the IdP without an SP AuthnRequest. The IdP posts a Response directly to the SP's ACS; simpler to start but offers fewer request-bound security guarantees.

<!-- longDescription:end -->
