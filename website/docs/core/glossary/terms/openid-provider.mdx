termName: OpenID Provider (OP)

<!-- shortDescription:begin -->

OIDC authority that authenticates users and issues tokens.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Authorization Server implementing OpenID Connect. Exposes discovery metadata, authorization, token, userinfo, JWKS, and end-session endpoints; authenticates users, issues ID/Access/Refresh tokens, and enforces consent and policy. In OAuth2 terminology, the OP is the Authorization Server (AS).

<!-- longDescription:end -->
