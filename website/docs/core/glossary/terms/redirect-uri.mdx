<!-- shortDescription:begin -->

Callback URL the provider redirects to.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Registered callback URL where the OpenID Provider (OP) or Authorization Server (AS) returns the user after authorization. Must match exactly (including scheme and path), and pairs with response_mode (query, fragment, or form_post).
