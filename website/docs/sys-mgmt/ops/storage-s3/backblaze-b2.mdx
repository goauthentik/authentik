## Create the bucket in Backblaze

1. Log in to the [Backblaze web console](https://secure.backblaze.com/user_signin.htm).
2. Go to **B2 Cloud Storage** > **Buckets**.
3. Click **Create a Bucket**.
4. Enter `<bucket_name>` as the bucket name.
5. Set **Files in Bucket are** to **Private**.
6. Click **Create a Bucket**.
7. Open the bucket details and note the bucket region.

## Create Backblaze credentials

1. In the Backblaze web console, go to **B2 Cloud Storage** > **Application Keys**.
2. Click **Add a New Application Key**.
3. Enter a key name, for example `authentik-s3`.
4. Restrict the key to the bucket that authentik uses.
5. Grant read and write access.
6. Click **Create New Key**.
7. Copy the key ID and application key.

Backblaze S3-compatible endpoints use the following format:

```text
https://s3.<region>.backblazeb2.com
```

## Configure authentik

Add the shared S3 settings, then set the Backblaze endpoint and region:

```env
AUTHENTIK_STORAGE__S3__ENDPOINT=https://s3.<region>.backblazeb2.com
AUTHENTIK_STORAGE__S3__REGION=<region>
```

Use the application key ID as `AUTHENTIK_STORAGE__S3__ACCESS_KEY` and the application key as `AUTHENTIK_STORAGE__S3__SECRET_KEY`.
