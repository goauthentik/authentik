termName: Distinguished Name (DN)

<!-- shortDescription:begin -->

Unique path identifying an entry in LDAP.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Hierarchical identifier built from Relative Distinguished Names (RDNs), e.g., 'uid=jane,ou=People,dc=example,dc=com'. Used to reference and bind to entries.

<!-- longDescription:end -->
