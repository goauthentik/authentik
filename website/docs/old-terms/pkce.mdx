<!-- shortDescription:begin -->

Proof Key for Code Exchange hardens the code flow.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Binds the authorization request to the token exchange using a one-time code verifier and a code challenge (typically S256). Prevents intercepted codes from being redeemed by attackers.

<!-- longDescription:end -->
