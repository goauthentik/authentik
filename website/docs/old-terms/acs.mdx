termName: Assertion Consumer Service (ACS)

<ShortDescription>SP endpoint that receives SAML assertions.</ShortDescription>

<LongDescription>
    Configured SP endpoint where the IdP delivers the SAML Response (typically via HTTP-POST). The
    SP validates signatures, issuer, audience, and time conditions before creating a session.
</LongDescription>
