<!-- shortDescription:begin -->

Endpoint to validate opaque tokens.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

RFC 7662 endpoint that returns whether a token is active along with subject, scopes, audience, and expiry. Typically requires client authentication and is used when tokens are opaque.

<!-- longDescription:end -->
