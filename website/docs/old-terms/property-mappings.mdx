<!-- shortDescription:begin -->

Define how data is exposed to apps and stored from sources.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Property Mappings allow you to make information available for external applications and to modify how information from sources is stored in authentik. For example, to log in to AWS you can set a user's roles in AWS based on their group memberships in authentik.

<!-- longDescription:end -->
