<!-- shortDescription:begin -->

OAuth2 role that issues tokens and hosts auth endpoints.

<!-- shortDescription:end -->
<!-- longDescription:begin -->

Server that authenticates resource owners, obtains consent, and issues access/refresh (and in OIDC, ID) tokens to clients. Provides authorization, token, introspection, revocation, and metadata (discovery) endpoints. In OIDC, the Authorization Server is called the OpenID Provider (OP).

<!-- longDescription:end -->
