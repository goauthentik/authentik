## What does it do

Allows you to SSH to Linux hosts using authentik credentials.

## Platform components on Linux

### authentik-cli

Provides commands for interacting with authentik-agent.

### authentik-agent

User service.

### authentik-sysd

System service.

### libpam-authentik

PAM Module. Requires authentik-sysd running.

### libnss-authentik

NSS Module. Requires authentik-sysd running.
