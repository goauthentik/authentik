# Development environment setup

This section provides guides for setting up your development environment to work on different components of authentik.

## Getting started

Based on what part of authentik you want to work on, you'll need to set up different environments:

- [Frontend development environment](./envs/frontend.md) - For working on the web UI
- [FUll development environment](./envs/full.mdx) - For backend and frontend development
- [Website/documentation development](./envs/website.md) - For working on the documentation
- [Outpost development](./envs/outposts.md) - To develop outposts

## Common development workflows

- **Making a change to the UI**: Set up the frontend environment and run the dev server
- **Adding a new API endpoint**: Set up the backend environment and implement the endpoint in Django
- **Fixing a documentation issue**: Set up the documentation environment and make your changes
- **Adding a feature to an outpost**: Set up the outpost development environment

See the specific setup guides for detailed instructions on each development workflow.
